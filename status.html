Personal Budget Tracker API

API Status: Running

The API is operational and accepting requests.

Available API Routes

  • POST /api/auth/register - Register a new user
  • POST /api/auth/login - Authenticate and get token
  • GET /api/entries - Get all financial entries
  • POST /api/entries - Create a new financial entry
  • PUT /api/entries/:id - Update a financial entry
  • DELETE /api/entries/:id - Delete a financial entry
  • GET /api/categories - Get all categories
  • GET /api/recurring-payments - Get recurring payments
  • POST /api/recurring-payments - Create a recurring payment
  • PUT /api/recurring-payments/:id - Update a recurring payment
  • DELETE /api/recurring-payments/:id - Delete a recurring payment
